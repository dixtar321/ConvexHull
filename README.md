Инструкция
```bash
После запуска Task3.exe необходимо загрузить текстовый файл с координатами точек. После выбора - по точкам будут построены выпуклая и вогнутая оболочки. Коэффициент гамма (степень глубины) регулируется от 0.00 до 2.00 с шагом 0.10. После изменения коэффициента гамма и нажатия на кнопку "Рассчитать", вогнутая оболочка перестроится с учётом коэффициента. После можно сохранить результат по нажатии на одноимённую кнопку. Файл сохраняется в формате result_дата_время.txt и хранит в себе координаты точек охватывающего полигона.
```
